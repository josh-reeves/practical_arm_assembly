About

This site is meant to serve as documentation for the Practical Arm Assembly GitHub repository, which, in turn, is based on the Practical Arm Assembly Tutorial Series by Laurie Wired. Together, the repo and documentation are meant to serve as an introduction for anyone attempting to learn ARM assembly, with the repository providing well-commented code examples and the documentation providing supplemental, in-depth information about the topics covered.

Please note that this effort is not in any way affiliated with the creator of the original tutorial series and these materials are being provided for purely educational purposes. While the associated repository isn't officially licensed for a variety of reasons, all code, concepts and other intellectual property should be attributed to their original authors.



What is ARM Assembly?

Assembly languages are low-level programming languages that feature a strong correlation between the instructions the language provides and the machine code instructions the processor is capable of natively understanding. They provide a way to directly invoke the basic arithmetic and logic operations that the processor can perform.

Because these operations and the way they're performed vary between processor architectures, assembly languages are typically specific to a particular architecture. ARM Assembly is the language for interfacing with processors based on the ARM architecture. For more information, please refer to About ARM Assembly the section of the introduction.
