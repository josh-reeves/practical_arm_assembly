Procedures

Section 1 explained that ARM assembly code is organized into labels that reference data or instructions. These labels can be used to create procedures: Blocks of instructions that perform specific tasks. Like functions, procedures are designed to be modular and reusable.

add_nums:

r0, r1, r2 // Adds the values stored in r1 and r2 together and stores the result in r0.


bx lr// Returns execution to the address stored in lr.
