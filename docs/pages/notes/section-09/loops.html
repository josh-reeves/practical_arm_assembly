Loops

The previous section explored how the Branch instruction can be used for conditional execution. In addition to conditional statements, this can also be used to create loops. The below implements a simple while-loop with the Branch instruction:

_start:

cmp r0, #0x5

bge exit


add r0, r0, #0x1

b _start


exit:

mov r7, #0x1

svc 0

The program begins by comparing the value stored in register r0 to 5. The following line pairs the Branch instruction with the Greater than or Equal to (ge) mnemonic extension to restrict execution of the instruction based on the value stored in r0. If the value is greater than or equal to 5, execution will branch to the exit label and the program will end. Otherwise, 1 is added to the value currently stored in r0 and execution branches back to the beginning of the program. This is roughly equivalent to the following code in C:

_start:

cmp x0, #0x5

bge exit


add x0, x0, #0x1

b _start


exit:

mov x8, #0x5d

svc 0

The program begins by comparing the value stored in register r0 to 5. The following line pairs the Branch instruction with the Greater than or Equal to (ge) mnemonic extension to restrict execution of the instruction based on the value stored in x0. If the value is greater than or equal to 5, execution will branch to the exit label and the program will end. Otherwise, 1 is added to the value currently stored in x0 and execution branches back to the beginning of the program. This is roughly equivalent to the following code in C:

int i = 0;


while (i < 5)

{

i++;


}


return i;

Summary

In addition to conditional logic, the branch statement can be used to implement loops.
