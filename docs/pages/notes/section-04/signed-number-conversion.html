Negative Values in Binary

Understanding Two's Complement

As explained in in the notes for lesson 3, digital computers process all values in binary. To represent negative numbers these systems commonly use two's complement. System's that use two's complement interpret the most significant digit of a binary value as representative ove the value's sign, with 0s representing positive values and 1s representing negative values. The manner in which the remaining digits are parsed depends on the sign bit, with negative values "counting down" upward toward zero and positive values "counting up" as usual.

Summary

Signed values in binary are represented using two's complement. Systems that use two's complement interpret the most significant bit as the value's sign. The interpretation of the remaining bits is dependant on the sign bit. The two's complement of a binary value can be found by inverting all of the bits and adding 1 to the resulting value.
